# 优化建图

## 线段树优化建图

1. 对于批量的点来说，如果每个点都单独连边，复杂度无法接受，需要优化连边的方式，
2. 利用线段树结构组织区间，每次批量的点都拆分成 `log n` 个区间点，减少边的数量，
3. 小区间连向大区间用出树，大区间连向小区间用入树，两棵树的叶节点可以共用，
4. 区间向单点连边的过程，
5. 单点向区间连边的过程，
6. 区间向区间连边的过程。

## 树链剖分优化建图

1. 树链剖分的便利性，重链上的节点 `dfn` 序号连续，子树上的节点 `dfn` 序号连续，
2. 子树节点批量连边，使用线段树即可，
3. 路径节点批量连边，树剖跳转加线段树更新，
4. 空间 `O(n)` ，单次操作时间 `O(log n * log n)` ，常数时间较小。

## 倍增优化建图

1. 路径节点批量连边的操作，只能接受 `O(log n)` 的复杂度，
2. 每个节点有 `log n` 个区间点，表示节点向上的范围节点，用倍增表记录，
3. 底层小范围连向高层大范围用出表，高层大范围连向底层小范围用入表，最底层的单点编号共用，
4. 区间向单点连边的过程，
5. 单点向区间连边的过程，
6. 区间向区间连边的过程，
7. 空间 `O(n * log n)` ，单次操作时间 `O(log n)` ，常数时间较大。

## 2-AST与前缀优化建图

### 2-SAT问题与求解过程

1. 2-SAT（Satisfiability）问题：每个变量的取值只有 `0` 和 `1` 两种，为了满足所有给定的限制，确定每个变量的值。
2. 每个变量 `x` ，有两个互斥的取值假设， `x` 和 `¬x` 。
3. 每个限制，形如 `x` 或 `y` ，转化为两条关系， `¬x -> y` 和 `¬y -> x` 。
4. 每个限制转化为关系时，要满足取反反向的对称性，若 `u -> v` 必有 `¬v -> ¬u` 。
5. 每个取值假设认为是点，每条关系认为是边，就构成了图，求解图中的强连通分量。
6. 如果某个变量的互斥取值假设，对应图中的两个点，在同一个强连通分量里，则无解。
7. 否则有解，每个变量选择SCC缩点后的DAG中，拓扑序更靠后的取值假设，作为变量的值。
8. 变量取值时注意，拓扑序越靠后的点，归属的强连通分量的序号越小。

### 例子1

- 根据如下两个限制，求出一组 `x` 和 `y` 的真假状况。
- `x == 1` 或 `y == 1` 。
- `x == 1` 或 `y == 0` 。

### 例子2

- 给定若干变量和限制，建好图之后，求解了强连通分量。
- 变量 `x` ，互斥取值假设为 `a` 和 `b` ， `a` 的拓扑序小于 `b` 的拓扑序。
- 变量 `y` ，互斥取值假设为 `c` 和 `d` ， `c` 的拓扑序小于 `d` 的拓扑序。
- 会不会出现如下情况，导致生成答案的策略失效？
- 根据拓扑序，变量 `x` 取值为 `b` ，进行推导发现，变量 `y` 被迫取值为 `c` 。
- 不会出现这种情况，因为取反反向的对称性。

### 前缀优化建图的原理

1. 线段树优化建图，任意区间的每个点，进行快速连边，时间复杂度 `O(log n)` ，
2. 支持前缀区间，设计更简单，时间复杂度 `O(1)` ，
3. 前缀区间 `[1, x]` 的每个点向单点y连边，
4. 单点 `y` 向前缀区间 `[1, x]` 的每个点连边，
5. 前缀区间 `[1, x]` 的每个点向前缀区间 `[1, y]` 的每个点连边，
6. 后缀优化建图同理。
