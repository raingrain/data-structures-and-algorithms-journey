# 矿场搭建

## [P3225 [HNOI2012] 矿场搭建](https://www.luogu.com.cn/problem/P3225)

> - **Question**
>   - 一共 `n` 个地点，地点至少 `2` 个，每个地点都有人， `m` 条双向道路连通所有地点，地震会发生在任何一个地点，地震发生时，其他地点的人都要去往救援点，你可以在任何地点设立救援点，但是发生地震的地点，道路和救援点都会失效，打印至少需要几个救援点，打印设立救援点的方案总数，方案认为是无序集合。
>   - **Tips**
>     - `1 <= n <= 1000`
>     - `1 <= m <= 1000`
