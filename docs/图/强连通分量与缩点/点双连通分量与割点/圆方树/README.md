# 圆方树（block cut tree）

1. 代表原图的点和点双连通分量的关系；
2. Tarjan算法求解点双连通分量的过程，可以用来建立圆方树；
3. 原图的点为圆点，点双连通分量为方点；
4. 节点数量为 `n` ，圆点编号 `<= n` ，方点编号 `> n` ；
5. 圆方树中的圆点和方点交替出现，圆点的邻居都是方点，方点的邻居都是圆点；
6. 圆点的每个邻居都是当前点所归属的点双连通分量；
7. 方点的每个邻居都是当前点双连通分量所拥有的点；
8. 圆点只有一个邻居，该点不是割点，多于一个邻居，该点是割点；
9. 原图中任意两点之间的必经点等于圆方树简单路径中的圆点；
10. 一个点双连通分量的子图内，任意三点 `a, b, c` ，存在 `a` 到 `b` ，然后 `b` 到 `c` 的路径，沿途节点不重复。
