# LCA

## 各种求解LCA的算法对比

1. 树上倍增，预处理的空间和时间 `O(n * log n)` ，单次查询 `O(log n)` 。
2. `tarjan` 算法，预处理的空间和时间 `O(n + m)` ，单次均摊 `O(1)` ，只能处理批量离线任务。
3. 重链剖分，预处理的空间和时间 `O(n)` ，单次查询 `O(log n)` 。
4. 欧拉序结合倍增表，预处理的空间和时间 `O(n * log n)` ，单次查询 `O(1)` 。
5. `dfn` 序结合倍增表，预处理的空间和时间 `O(n * log n)` ，单次查询 `O(1)` ，常数是欧拉序的一半。

以上总结都是针对静态树
