# [交换数字](https://leetcode-cn.com/problems/swap-numbers-lcci/)

### 解题思路
1. 一道异或运算比较经典的题目，如何不用临时变量交换两个数，分三步:
2. a = a ^ b
3. b = a ^ b
4. a = a ^ b
5. 注意，a与b不能对应同一块内存区域

### 代码

```java
// java
class Solution {
    public int[] swapNumbers(int[] numbers) {
        numbers[0] = numbers[0] ^ numbers[1];
        numbers[1] = numbers[0] ^ numbers[1];
        numbers[0] = numbers[0] ^ numbers[1];
        return numbers;
    }
}
```
